# Bee's Beans & Greens

A mobile-first gothic coffee and tea ordering menu.

## Included

- Dark Victorian / macabre visual theme
- Custom gothic bee crest artwork
- Landing screen
- Complete Beans and Greens menu
- Drink modifier dropdowns
- Working shopping cart
- Mobile responsive design
- GitHub Pages compatible

## Publish on GitHub Pages

Upload these four files to the root of your repository:

- `index.html`
- `style.css`
- `script.js`
- `menu.json`

Then open **Settings → Pages**, choose **Deploy from a branch**, select `main` and `/ (root)`, then save.

## Current checkout behavior

The cart is functional, but the checkout button is intentionally a demo. Secure payments require Stripe Checkout or another hosted payment provider. Do not put Stripe secret keys in these public files.

## Next phase

Connect a hosted Stripe Checkout flow, order collection, and notifications through a secure backend or automation service.
